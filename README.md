# Enquete

Apenas um formulário para saber qual app as pessoas gostariam de terem hoje em dia.

# Dependências
- **ejs**
- **express**
- **sqlite**
- **sqlite3**
- **eslint**
- **terser**
- **dotenv**

# Iniciando o Projeto em dev

npm run dev

# Iniciando em produção

- **Primeiro cria o arquivo .env e coloca em production**

# Envs
 - **PORT**=SUA PORTA
 - **NODE_ENV**=production

# Fazendo o deploy

npm run build

...

npm start
